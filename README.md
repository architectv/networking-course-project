## Краткий перечень функциональных требований

* Регистрация.
* Авторизация.
* Просмотр, создание, редактирование, удаление:
  * проекты;
  * канбан-доски;
  * списки задач;
  * задачи;
  * метки к задчам;
  * права участников.
